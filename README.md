# Number Guessing Game

A simple and engaging web-based Number Guessing Game built using HTML, CSS, and JavaScript. Players attempt to guess a randomly generated number within a limited number of attempts while receiving hints to guide them toward the correct answer.

## Live Demo

https://elegant-guess-cc8946.netlify.app/

## Features

* Random number generation
* Interactive user interface
* Instant feedback on guesses
* Hint system (Too High / Too Low)
* Score tracking
* Restart game functionality
* Responsive design

## Technologies Used

* HTML5
* CSS3
* JavaScript

## How to Play

1. Enter a number in the input field.
2. Submit your guess.
3. Receive feedback indicating whether your guess is too high or too low.
4. Continue until you find the correct number.
5. Try to win in the fewest attempts possible!
